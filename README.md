# Barani Portal

A full-stack financial management portal developed to streamline internal business operations and employee workflows securely. The system provides centralized administrative control for managing employees, credentials, and financial records efficiently.

---

## Features

- Secure admin-controlled authentication
- Employee and manager account management
- Financial record and data tracking
- Dashboard for internal operations
- Database-driven workflow management
- Responsive frontend interface
- Secure credential handling

---

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/RaghulSuriya/Barani-Portal.git
```

---

### 2. Move project to XAMPP htdocs

Copy the project folder into:

```bash
C:\xampp\htdocs
```

---

### 3. Start XAMPP Services

Start:
- Apache
- MySQL

---

### 4. Configure Database

- Open phpMyAdmin
- Create database
- Import SQL file

---

### 5. Run Frontend

```bash
npm install
npm start
```
